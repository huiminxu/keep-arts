## ARTS
记录每周践行ARTS的成长。

### 什么是ARTS

每周至少做一个leetcode的算法题、阅读并点评至少一篇英文技术文章、学习至少一个技术技巧、分享一篇有观点和思考的技术文章。（也就是Algorithm、Review、Tip、Share简称ARTS）
